Discuz! 抽奖程序
=====================


#### 说明

事实上它不是一个标准的DZ插件，因为没有按照标准的DZ插件写法来写
不过可以基本实现操纵积分的目的

#### 使用方法

直接把这个文件移动到主目录下即可
